# AquaFizz Employee Compliance and Sales Analysis

## Project Overview
This project analyzes employee working hours for compliance with company policy and provides sales insights using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- VBA (Macro for Reset Dashboard)
- Charts and Slicers

## Key Features
- Employee working hours compliance check
- Automated non-compliance messages
- Sales and profit analysis
- Interactive Excel dashboard
- Reset button using VBA macro
